# Codewars Rust Solutions

This repository contains my solutions to algorithmic challenges on Codewars, solved using the Rust programming language.

The solutions are organized by difficulty rank (Kyu), following the Codewars ranking system. The structure progresses from 8 Kyu (Introductory) towards 1 Kyu (Mastery).

### Purpose
The primary goal of this repository is to document my learning journey and demonstrate objective progress. It serves as an archive to observe how my problem-solving approach and understanding of Rust idioms evolve over time.

### Structure
* 8kyu/ - Fundamentals
* 7kyu/ - Beginner
* 6kyu/ - Novice
* 5kyu/ - Intermediate
